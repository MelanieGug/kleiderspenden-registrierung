Datenschutzerklärung

Diese Website ist ein Studienprojekt („MG Kleiderspendenportal“) und dient ausschließlich der Demonstration einer Webanwendung. Der Schutz deiner personenbezogenen Daten ist uns wichtig. Im Folgenden informieren wir darüber, welche Daten auf dieser Website verarbeitet werden und zu welchem Zweck.

Welche Daten werden verarbeitet?

1) Daten aus dem Formular (Kleiderspenden-Registrierung)

  • Art der Kleidung
  • ausgewähltes Krisengebiet
  • nur bei Abholung: Straße/Hausnummer, Postleitzahl, Ort

2) Technische Daten (automatisch beim Aufruf)

Beim Aufruf einer Website werden durch deinen Browser technisch bedingt Daten übertragen (z. B. IP-Adresse, Datum/Uhrzeit, Browsertyp). Diese Website wertet solche Daten nicht aktiv aus und speichert sie nicht selbst in einer eigenen Datenbank.

Zweck der Verarbeitung

Die Eingaben werden ausschließlich verwendet, um die Registrierung zu prüfen (Pflichtfelder & PLZ-Regel) und eine Bestätigungsseite mit den eingegebenen Informationen anzuzeigen.

Speicherung der Daten

Diese Website speichert die Formulardaten nicht auf einem Server. Die Daten werden nur vorübergehend im Browser gespeichert, damit sie auf der Bestätigungsseite angezeigt werden können.

  • Verwendete Technik: sessionStorage
  • Speicherdauer: bis du den Tab/Browser schließt oder eine neue Registrierung startest
  • Zweck: Anzeige der Bestätigung nach dem Absenden des Formulars

Weitergabe an Dritte

Eine Weitergabe der Formulardaten an Dritte findet nicht statt.

Hosting / GitHub Pages

Diese Website wird über GitHub Pages bereitgestellt. Beim Aufruf der Seiten verarbeitet GitHub technisch bedingt Verbindungsdaten (z. B. IP-Adresse), um die Inhalte auszuliefern.

Cookies und Tracking

  • keine Cookies zu Tracking- oder Werbezwecken
  • keine Analyse-Tools (z. B. Google Analytics)
  • keine externen Tracking-Skripte

Da im Rahmen dieses Studienprojekts keine serverseitige Speicherung deiner Formulardaten erfolgt, können in der Regel keine personenbezogenen Daten aus dem Formular nachträglich zugeordnet oder herausgegeben werden.
